# Norwood Ready Kids (NRK) Database
## Overview
Designed and implemented an 11-table relational database to streamline performance tracking for early education programs. This project focuses on data integrity, efficient schema design, and optimized query workflows.

## Tools & Techniques
- **Tools:** SQL (MySQL/SQL Server), Lucidchart, SQL Fiddle
- **Techniques:** Entity-Relationship Modeling (ERD), Relational Schema Design, Complex Query Optimization

## Key Findings
- Successfully normalized data structures, reducing redundancy and improving data retrieval speed for student assessment metrics.
- Developed verified, high-performance queries to support administrative reporting workflows.

## Repository Contents
- `/schema`: Contains the SQL script for database construction.
- `/queries`: Contains optimized SQL queries for reporting and analysis.
